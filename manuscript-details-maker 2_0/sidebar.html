Manuscript Details Maker — Guide

Getting Started

  • Fill in SHE name and Journal. Date auto-fills each time the page loads. — these appear in all copies.
  • Complete the Submission Checks in order. Optional notes/Concerns can be added even if boxes aren’t checked or dropdown options unselected.
  • Additional Statements: Tick relevant boxes or add note if there are concerns. If statements are unavailable, click ❌.
  • Peer Review: Add rounds and reviewer details as needed.
  • Final Decision: Write summary notes or decision comments.

Copying Data

  • Each section has its own Copy button.
  • Copy All (combined) gathers everything (submission, reviews, decision).
  • A short “Copied!” message confirms success.
  • The boxes or section content you don't fill do not get copied.

Resetting

  • Reset (preserve Name/Journal/Date) clears all other fields.
  • Useful for handling multiple manuscripts for the same SHE/journal.

Tips

  • Keep notes short and factual.
  • Use Opinion for overall remarks.
  • Potential Reviewer Expertise = key topics reviewers should have.
  • Refresh clears all data (tool doesn’t autosave).
